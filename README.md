## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Run Test Suite:  

`npm test`  
To Run E2E cypress test:  

`npm run cypress`  

To Start Server:

`npm start`  

To Visit App:

`localhost:3000`
